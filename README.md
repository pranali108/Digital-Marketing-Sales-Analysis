# Digital-Marketing-Sales-Analysis
 Project Description  This project analyzes digital marketing campaign performance across multiple advertising platforms using Power BI. The dashboard provides insights into sales performance, campaign effectiveness, platform profitability, and monthly trends to support data-driven marketing decisions.

## Overview
This project analyzes the performance of digital marketing campaigns run by an e-commerce company across multiple advertising platforms. The goal is to evaluate campaign effectiveness, identify profitable platforms, and uncover insights that help maximize revenue from digital marketing efforts.

An interactive dashboard was built using Power BI to visualize sales performance, campaign results, platform profitability, and monthly trends.

---

## Business Problem
A multinational e-commerce company runs digital marketing campaigns across platforms such as Google Ads, Facebook Ads, and YouTube Ads. Each platform uses different ad formats including Display Ads, Overlay Ads, Skippable Ads, Non-Skippable Ads, and Bumper Ads.

The marketing team wants to understand:

- Which platforms generate the highest sales
- Which campaigns perform best
- Whether sales targets are achieved
- How sales and profit vary across time
- Which marketing efforts deliver the best ROI

---

## Dataset
The dataset contains **190+ campaign performance records** with key marketing and sales metrics.

### Key Features
- Platform (Google Ads, Facebook Ads, YouTube Ads)
- Campaign Name
- Ad Format
- Impressions
- Clicks
- Conversions
- Average Pages Visited
- Average Time on Site
- Quantity Sold
- Unit Cost Price
- Unit Sale Price
- Final Cost Price
- Final Sale Price
- Target Sales
- Month / Date

This dataset enables analysis of campaign performance, customer engagement, sales, and profitability.

## Project Workflow

### 1. Data Loading
Imported the dataset into Power BI using the Get Data feature.

### 2. Data Cleaning
- Verified data types
- Checked for missing values
- Standardized numerical fields

### 3. Data Transformation
Created calculated measures including:
- Total Sales
- Total Profit
- Sales vs Target
- Conversion Performance

### 4. Dashboard Development
Built an interactive Power BI dashboard to visualize campaign performance, platform effectiveness, and sales trends.

## Dashboard Insights

### Sales Performance
- Compared total sales across marketing platforms.
- Identified platforms generating the highest revenue.

### Campaign Effectiveness
- Analyzed campaign-level sales and conversion performance.
- Highlighted top-performing marketing campaigns.

### Profitability Analysis
- Compared cost price and sale price to calculate profit.
- Identified platforms delivering the highest profitability.

### Sales Target Analysis
- Evaluated actual sales against target sales to measure campaign success.

### Monthly Trends
- Analyzed monthly sales fluctuations to understand marketing performance over time.

### Quantity Analysis
- Evaluated product demand using quantity sold trends.
